# Terant: the new termux-ant.

--------------------------------------------

## 使用方法：


1. 同步脚本
````sh
$ git clone https://github.com/urain39/terant
````

2. 进入目录
````sh
$ cd terant
````
3. 切换到root
````sh
$ tsu -p #注意加上p参数
````
4. 安装（以root执行）
````sh
# bash terant.sh # 本人只保证bash能执行。
````
5. 启动系统（以root执行）
````sh
# terant
````
----------------------------------------------

注意事项：

    1. 可能需要执行两次脚本才能生效
    
    2. 卸载请运行uninst.sh脚本
    
    3. root版本需要使用tsu -p手动进入roo后才能使用
